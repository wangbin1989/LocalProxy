# v0.0.4 — 简化 run 命令

## 概述

移除 `run` 命令的单代理 CLI 模式（`-l`/`-H`/`-p`/`-P`），统一通过 `--config` 配置文件启动代理。

## 变更内容

### 移除

- `run` 命令的单代理 CLI 选项（`-l`、`-H`、`-p`、`-P`）
- `RunHandler.Handle` — 单代理启动方法
- `ConsoleOutput.ShowStartupPanel` — 单代理面板

### run 命令

```
localproxy run --config <file>
```

`--config` 为必填选项，指定 JSON 配置文件路径。文件中的所有代理将被启动。
