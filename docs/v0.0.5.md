# v0.0.5 — `--config` 默认配置文件路径

## 概述

`--config` 选项改为非必填，默认使用 `~/.localproxy.json` 作为配置文件地址。

## 变更内容

### 新增

- `ConfigService.DefaultConfigPath` — 默认配置文件路径，指向 `~/.localproxy.json`
- `ConfigServiceTests.DefaultConfigPath_ReturnsPathUnderUserProfile` — 新增测试

### 修改

- `run --config` 和所有 `config` 子命令（`add`/`update`/`remove`/`enable`/`disable`/`list`）的 `--config` 选项均改为非必填，通过 `DefaultValueFactory` 设置默认值

### 使用示例

```bash
# 省略 --config，自动使用 ~/.localproxy.json
localproxy run
localproxy add my-proxy -l 3306 -H db.internal -p 3306 -P tcp
localproxy list

# 仍可显式指定配置文件
localproxy run --config /path/to/custom.json
```
