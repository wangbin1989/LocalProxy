## 用户故事

### GUI-006: 最小化到系统托盘（8 点）

> 作为桌面用户，我希望关闭窗口时最小化到系统托盘。

**验收标准：**
- 关闭窗口（点击 X）最小化到托盘而非退出
- 系统托盘图标可见
- 双击托盘图标恢复窗口
- 托盘图标右键菜单 → "退出" 才是真正退出
- 使用 `H.NotifyIcon` 库实现

**技术要点：**
- macOS：`H.NotifyIcon` 需要 `NSApplication` 事件循环
- Windows：托盘图标在任务栏通知区域
- 需要处理应用生命周期（MAUI 默认关闭=退出，需拦截）

### GUI-007: 托盘右键菜单（5 点）

> 作为用户，我希望系统托盘右键菜单有快捷操作。

**验收标准：**
- 右键菜单项：
  - 显示主窗口
  - 全部启动（启动所有已配置的隧道）
  - 全部停止（停止所有运行中的隧道）
  - 分隔线
  - 退出
- 操作结果通过通知气泡反馈（成功/失败）
- 全部启动/停止通过 IPC 批量操作

### GUI-008: 开机自启开关（8 点）

> 作为用户，我希望有开机自启开关。

**验收标准：**
- 设置面板有"开机自启"复选框
- 勾选后创建平台对应的自启条目：
  - **macOS：** 创建 `~/Library/LaunchAgents/com.localproxy.plist`
  - **Windows：** 写入 `HKCU\Software\Microsoft\Windows\CurrentVersion\Run`
  - **Linux：** 创建 `~/.config/autostart/localproxy.desktop`
- 取消勾选后删除对应条目
- 重启后验证自启生效

### GUI-009: 日志查看面板（5 点）

> 作为用户，我希望 GUI 中有日志查看面板。

**验收标准：**
- 日志面板显示最近 500 条日志
- 按级别过滤（Info / Warn / Error / All）
- 自动滚动开关
- "打开日志文件"按钮（调用系统默认应用打开）
- 日志列表虚拟化滚动（大量日志不卡顿）

### GUI-010: 快速设置向导（5 点）

> 作为用户，我希望首次启动有快速设置向导。

**验收标准：**
- 首次启动（无可用的自定义配置时）自动弹出向导
- 向导步骤：
  1. 选择预置模板（Redis、SQL Server 等，含图标和说明）
  2. 输入远程主机地址
  3. 确认并创建代理
- 仅 3 步，2 分钟内完成
- 向导完成后代理自动启动
- 可跳过向导，稍后手动配置
