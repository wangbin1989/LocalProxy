# Sprint 7 — MAUI GUI 基础

**周期：** 第 14-15 周（2 周）  
**目标：** 构建 .NET MAUI 桌面应用核心界面，包括代理列表、增删改表单和实时统计

---

## 冲刺目标

实现 `LocalProxy.App` MAUI 项目，使用 MVVM 架构提供可视化的代理管理界面，通过 IPC 与后台服务通信。

---

## 用户故事

### GUI-001: 代理列表（8 点）

> 作为桌面用户，我希望看到所有代理列表及状态指示。

**验收标准：**
- 主窗口以列表/表格展示所有代理
- 每行显示：名称、协议、本地端口、远程地址、状态（彩色指示灯：绿=运行、灰=停止、红=错误）、启停按钮
- 列表从后台服务通过 IPC 获取数据
- 每 2 秒自动刷新状态
- 支持键盘导航（上下箭头切换选中行）

### GUI-002: 添加代理表单（8 点）

> 作为用户，我希望通过表单对话框添加新代理。

**验收标准：**
- "添加代理"按钮打开表单对话框
- 表单字段：名称、协议（下拉选择 TCP/UDP/HTTP）、本地端口、远程主机、远程端口、最大连接数（默认 100）、超时秒数（默认 30）
- 内联校验：端口范围、名称非空、主机格式
- 保存后写入配置文件，列表自动刷新
- 从预置模板快速创建（下拉选择模板，自动填充）

### GUI-003: 编辑和删除代理（5 点）

> 作为用户，我希望从 GUI 编辑和删除已有代理配置。

**验收标准：**
- 编辑按钮打开预填充的表单
- 删除按钮弹出确认对话框
- 操作结果持久化到配置文件
- 列表即时反映变更

### GUI-004: 一键启停（5 点）

> 作为用户，我希望从列表一键启动/停止每个代理。

**验收标准：**
- 每行有启动/停止按钮（根据当前状态切换）
- 点击后通过 IPC 通知服务执行操作
- 状态指示实时更新
- 操作失败时弹出错误提示

### GUI-005: 实时统计面板（5 点）

> 作为用户，我希望在 GUI 中看到实时连接统计。

**验收标准：**
- 选中运行中的代理，显示统计面板
- 统计项：收发字节数、活跃连接数、运行时长
- 每 2 秒自动刷新
- 字节数自动格式化（KB/MB/GB）

### GUI-011: 暗色模式（3 点）

> 作为用户，我希望 GUI 支持暗色模式。

**验收标准：**
- 使用 MAUI 内置主题系统
- 跟随系统设置自动切换（也可手动切换）
- 所有页面在亮/暗模式下渲染正确
